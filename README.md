# دانلودر پلی‌لیست ویدیو

این ریپازیتوری با استفاده از GitHub Actions پلی‌لیست‌ها را دانلود می‌کند.

## نحوه استفاده

1. به تب **Actions** در GitHub بروید
2. روی **Download Playlist Videos** کلیک کنید
3. روی **Run workflow** کلیک کنید
4. آدرس پلی‌لیست و کیفیت مورد نظر را وارد کنید
5. کلیک **Run workflow**

## محدودیت‌ها

- هر بار اجرا حداکثر 6 ساعت زمان دارد
- فضای ذخیره‌سازی محدود (حداکثر چند گیگابایت)
- ویدیوهای بزرگ را به صورت تکی دانلود کنید

## دانلود خودکار

می‌توانید با تنظیم `schedule` در فایل workflow، دانلود خودکار را برنامه‌ریزی کنید.
